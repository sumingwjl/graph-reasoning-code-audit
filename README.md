## 🛠️ 依赖工具

本项目依赖以下开源工具，请确保在运行前完成安装：

| 序号 | 工具 | 仓库 | 用途 |
|:----:|------|------|------|
| 1 | **Graphify** | [safishamsi/graphify](https://github.com/safishamsi/graphify) | 将代码库构建为可查询的知识图谱 |
| 2 | **Semgrep** | [semgrep/semgrep](https://github.com/semgrep/semgrep) | 轻量级静态分析，支持 30+ 语言 |
| 3 | **CodeQL Action** | [github/codeql-action](https://github.com/github/codeql-action) | GitHub 官方 CodeQL 扫描 Action |
| 4 | **Joern** | [joernio/joern](https://github.com/joernio/joern) | 基于代码属性图（CPG）的静态分析框架 |
| 5 | **Betterleaks** | [betterleaks/betterleaks](https://github.com/betterleaks/betterleaks) | 密钥与敏感信息泄露扫描 |
| 6 | **OSV-Scanner** | [google/osv-scanner](https://github.com/google/osv-scanner) | Google 开源漏洞扫描器 |
